Get started with Ionic Framework

Start building high-quality mobile apps with the tech you know and love.

Welcome to the Ionic v2 Getting Started guide! Looking for our v1 Getting Started guide? Head over here.

  • Install Ionic

    {% highlight bash %}npm install -g cordova ionic{% endhighlight %}

    Windows Developers: We recommend Visual Studio Community, which comes with everything you need, including starter templates!

    First, install Node.js. Then, install the latest Cordova and Ionic command-line tools in your terminal. Follow the Android and iOS platform guides to install required tools for development.

    If you’re new to the command line, read our Terminal tutorial

  • Start a project

    {% highlight bash %}ionic start --v2 myApp tabs{% endhighlight %}

    Create an Ionic project using one of our ready-made app templates, or a blank one to start fresh. Check out the Market for more designs.

    • $ ionic start myApp blank
    • $ ionic start myApp tabs
    • $ ionic start myApp sidemenu

  • Run it

    {% highlight bash %} cd myApp ionic platform add ios ionic build ios ionic emulate ios {% endhighlight %}

    Create an Ionic project using one of our ready-made app templates, or a blank one to start fresh.

    Try Ionic View app to preview, test, and share your app, live on the device

That’s it. You’re done.
