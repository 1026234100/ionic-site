<h1 class="api-title">


Button






</h1>






<h2>Description</h2>

<p>Buttons are simple components in Ionic, can consist of text, an icon, or both, and can be enhanced with a wide range of attributes.</p>


<h2>Directive</h2>
<h3>selector: <code>button,[button]</code></h3>
