Icon

Description

Icons can be used on their own, or inside of a number of Ionic components. For a full list of available icons, check out the Ionicons resource docs.

Directive

selector: icon

Attributes:

Attribute Type Description
is-active boolean Whether or not the icon is active. Icons that are not active will use an outlined version of the icon. If there is not an outlined version for the particular icon, it will use the default (full) version.
ios string Explicitly set the icon to use on iOS.
md string Explicitly set the icon to use on Android.
