Slides

Description

Slides is a slide box implementation based on Swiper.js

Swiper.js: The most modern mobile touch slider and framework with hardware accelerated transitions

http://www.idangero.us/swiper/

Copyright 2015, Vladimir Kharlampidi The iDangero.us http://www.idangero.us/

Licensed under MIT

Component

selector: ion-slides

Usage

@Page({
 template: `
        <ion-slides pager (slide-changed)="onSlideChanged($event)" loop="true" autoplay="true">
         <ion-slide>
           <h3>Thank you for choosing the Awesome App!</h3>
           <p>
             The number one app for everything awesome.
           </p>
         </ion-slide>
         <ion-slide>
           <h3>Using Awesome</h3>
            <div id="list">
              <h5>Just three steps:</h5>
              <ol>
                <li>Be awesome</li>
                <li>Stay awesome</li>
                <li>There is no step 3</li>
              </ol>
            </div>
         </ion-slide>
         <ion-slide>
           <h3>Any questions?</h3>
         </ion-slide>
       </ion-slides>
       `
})

Attributes:

Attribute Type Description
autoplay Any whether or not the slides should automatically change
loop Any whether the slides should loop from the last slide back to the first
bounce Any whether the slides should bounce
pager add this property to enable the slide pager
slideChanged Any expression to evaluate when a slide has been changed
