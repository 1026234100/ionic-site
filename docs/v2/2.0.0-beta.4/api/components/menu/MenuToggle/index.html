MenuToggle

Improve this doc

The menuToggle directive can be placed on any button to automatically close an open menu.

Directive

selector: [menuToggle]

Usage

<button menuToggle>Toggle Menu</button>

To toggle a certain menu by its id or side, give the menuToggle directive a value.

<button menuToggle="right">Toggle Right Menu</button>
