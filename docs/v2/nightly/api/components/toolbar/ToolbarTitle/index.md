<p><code>ion-title</code> is a component that sets the title of the <code>ionToolbar</code> or <code>ionNavbar</code></p>

  &lt;ion-title&gt;SubHeader&lt;/ion-title&gt;
&lt;/ion-toolbar&gt;
</code></pre>
