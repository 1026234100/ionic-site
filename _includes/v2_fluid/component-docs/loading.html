Loading

Improve this Doc The Loading component is an overlay that prevents user interaction while indicating activity. By default, it shows a spinner whose styling is based on the platform mode (iOS, Material Design, Windows Universal). The spinner can be hidden or customized to use several predefined options, as well as dynamic content. The loading component is presented on top of other content even during navigation. _For complete usage information, check out the [Loading API docs](../api/components/loading/LoadingController)._

Basic Usage

Demo Source ```typescript import { LoadingController } from 'ionic-angular'; export class MyPage { constructor(public loadingCtrl: LoadingController) { } presentLoading() { let options = { content: "Please wait...", duration: 3000 } let loader = this.loadingCtrl.create(options); loader.present(); } } ```
