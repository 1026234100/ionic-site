Menu

Improve this Doc The Menu component provides a side-menu that is displayed by being dragged out or toggled, and can contain its own header and content. The content of a menu is hidden when the menu is closed. The styling of the Menu component automatically adapts to the platform the app is running on. _For complete usage information, check out the [API docs](../api/components/menu/Menu)._
