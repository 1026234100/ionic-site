Modal

Improve this Doc The Modal component slides a temporary view in from off screen. The content of a modal is provided by a separate component, with its own class and template that is imported into the component where the Modal will be created and displayed. Modals are often used for login, detail, or signup pages, message composition, and option selection. _For complete usage information, check out the [API docs](../api/components/modal/ModalController)._

Basic Usage

Demo Source ```typescript import { ModalController } from 'ionic-angular'; import { ModalPage } from './modal-page'; export class MyPage { constructor(public modalCtrl: ModalController) { } presentModal() { let modal = this.modalCtrl.create(ModalPage); modal.present(); } } ```
