Input

Improve this Doc The Input component implements the Angular 2 form module, and is essential for collecting and handling user input in a secure way. A variety of Input styles are available, and all follow the styling and interaction guidelines for each platform, which makes them intuitive for users to interact with. Each input field in a form has a **Control**, a function that binds to the value in the field and performs validation. A **Control Group** is a collection of Controls. Control Groups handle form submission, and provide a high level API that can be used to determine whether the entire form is valid. There are several types of Inputs available, each with its own behavior, including: - inline: label does not hide when text is entered - fixed: label is placed to the left of the input element - floating: labels animate or “float” up when the input is selected - inset: input spans less than 100% of the screen width - placeholder: input has placeholder text that disappears when text is entered - stacked: label always appears above the input, even after text is entered _For complete usage information, check out the [API docs](../api/components/input/Input)._
