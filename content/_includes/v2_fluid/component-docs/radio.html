Radio Button

Improve this Doc Like the [Checkbox](#checkbox) component, the Radio Button component is an Input component that holds a *boolean* value. Under the hood, Radio Buttons are no different than HTML radio buttons; however, like other Ionic components, radios are styled differently on each platform. Unlike the Checkbox component, Radio Buttons form a group, where only one Radio Button can be selected at a time. Use the `checked` input property to set the default value, and the `disabled` input property to disable the user from changing to that value. _For complete usage information, check out the [Radio Button API docs](../api/components/radio/RadioButton) and the [Radio Group API docs](../api/components/radio/RadioGroup)._
