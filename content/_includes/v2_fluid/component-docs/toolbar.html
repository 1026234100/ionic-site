Toolbar

Improve this Doc The Toolbar component is a generic bar that can be used in an app as a header, sub-header, footer, or even sub-footer. Since `ion-toolbar` is based on flexbox, no matter how many toolbars you have in your page, they will display correctly and `ion-content` will adjust accordingly. Toolbars can also contain other components, such as Buttons, Segment, Searchbar, and `ion-title`. Note: Typically a [NavBar](../api/components/navbar/Navbar) should be used inside `ion-header` for navigation, rather than a Toolbar. _For complete usage information, check out the [Toolbar API docs](../api/components/toolbar/Toolbar)._
