Toast

Improve this Doc Toast is a subtle notification that appears on top of an app's content. Typically, the Toast component is displayed for a short duration and then automatically dismissed. _For complete usage information, check out the [Toast API docs](../api/components/toast/ToastController)._

Basic Usage

Demo Source ```typescript import { ToastController } from 'ionic-angular'; export class MyPage { constructor(public toastCtrl: ToastController) { } presentToast() { let options = { message: 'User was added successfully', duration: 3000 } let toast = this.toastCtrl.create(options); toast.present(); } } ```
