Card

Improve this Doc Cards are a core user interface (UI) element for many apps, and are usually used to group related information. Cards are often stacked on top of one another in a list, but they can also be animated and swiped between like a page. _For complete usage information, check out the [Card API docs](../api/components/card/Card)._
