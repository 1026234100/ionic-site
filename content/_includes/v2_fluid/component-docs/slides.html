Slides

Improve this Doc Slides make it easy to create galleries, tutorials, and page-based layouts, and are implemented by nesting multiple `ion-slide` components inside an `ion-slides` compoentn. The behavior of Slides can be customized by binding an options object to the `options` input property of `ion-slides`. _For complete usage information, check out the [Slides API docs](../api/components/slides/Slides) and the [Slide API docs](../api/components/slides/Slide)._
