Toggle

Improve this Doc The Toggle component is an input component that holds a *boolean* value. Like the [Checkbox component](#checkbox), Toggles are often used to allow the user to switch a setting on or off. Attributes like `value`, `disabled`, and `checked` can be applied to the Toggle to control its behavior. _For complete usage information, check out the [Toggle API docs](../api/components/toggle/Toggle)._
