Input

Improve this Doc The Input component is used for collecting and handling user input in a secure way. A variety of Input styles are available, all of which follow the styling and interaction guidelines for each platform. Inputs are normally used in conjunction with the [Angular 2 Form module](https://angular.io/docs/ts/latest/guide/forms.html) to enable features like input validation and form submission. Ionic also provides a set of label styles for inputs, most of which are implemented using [utility attributes](): - **inline**: label does not hide when text is entered - **fixed**: label is placed to the left of the input element - **floating**: labels animate or “float” up when the input is selected - **inset**: input spans less than 100% of the screen width - **placeholder**: input has placeholder text that disappears when text is entered - **stacked**: label always appears above the input, even after text is entered _For complete usage information, check out the [Input API docs](../api/components/input/Input)._
