Input

Improve this Doc The Input component is used for collecting and handling user input in a secure way. A variety of Input styles are available, all of which follow the styling and interaction guidelines for each platform. Inputs are often used with the [Angular 2 Form module](https://angular.io/docs/ts/latest/guide/forms.html) to enable features like input validation and form submission. Ionic also provides a set of label styles/behaviors for Inputs, most of which are implemented using [utility attributes](): - **inline**: label doesn't hide when text is entered - **fixed**: label is placed left of the Input - **floating**: labels “float” up when Input is selected - **inset**: Input spans less than 100% of screen - **placeholder**: Input has placeholder text - **stacked**: label always appears above the input _For complete usage information, check out the [Input API docs](../api/components/input/Input)._
