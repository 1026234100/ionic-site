Tabs

Improve this Doc Tabs create a multi-tabbed interface with a Tab Bar and a set of views that can be tabbed through. Tabs are implemented by nesting multiple [`ion-tab`](../api/components/tabs/Tab/) components inside the [`ion-tabs`](../api/components/tabs/Tabs/) component. The view for each Tab is provided by a separate component, with its own class and template that is imported into the component where the Tabs will be implemented. Each Tab can be given a custom icon, badge, or text with the `tabIcon`, `tabBadge`, and `tabTitle` attributes. _For complete usage information, check out the [Tabs API docs](../api/components/tabs/Tabs) and the [Tab API docs](../api/components/tabs/Tab)._ ### Basic Usage Demo Source ```typescript import { Tab1 } from './tab1-page'; import { Tab2 } from './tab2-page'; @Component({ template: ` ` }) class MyApp { tab1: any = Tab1; tab2: any = Tab2; constructor() {} } ```
